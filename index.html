PawnIO

Download Now

PawnIO is a scriptable universal kernel driver, allowing hardware access to a wide variety of programs. It uses the Pawn scripting language and the PawnPP interpreter. When installed, any software implementing PawnIO modules can use the driver to access or control hardware supported by one of the official modules.

PawnIO comes in multiple distributions:

  • Official edition: The version available here. Identical to the open-source version, but digitally signed.
  • Unrestricted edition: An unsigned version without signature checking. Can be selected from the settings menu in the installer.
  • Open-source: The components PawnIO and PawnIOLib are made available under an open-source license that allows developers to create their own PawnIO forks.
  • Licensed: The PawnIO code is available for custom licensing. Please contact me for licensing options.
